General Routing

Path HTTP Methods Description
/ GET retrieve data from index.html

HTTP Methods for Users

Path HTTP Methods Description
/ GET Retrieves a list of users.
/register POST Creates a new user.
/update/:id PATCH Updates details of a specific user.
/users/:id DELETE Deletes a specific user.
/login POST Log user in.
/logout POST Log user out.

HTTP Methods for Products

Path HTTP Methods Description
/ GET Retrieves a list of shop.
/:id GET Retrieves details of a specific product.
/addProduct POST Creates a new product.
/updateProduct/:id PATCH Updates details of a specific product.
/delete/:id DELETE Deletes a specific product.
